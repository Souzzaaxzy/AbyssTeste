## 🛠 Instalação

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Souzzaaxzy/Abyss.git
   cd Abyss
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o bot:**
   ```bash
   npm start
   ```
